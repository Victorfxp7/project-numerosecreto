COMO JOGAR

Objetivo

Acertar o número inteiro que foi gerado aleatoriamente pelo algoritmo do programa no **menor número de tentativas possível**!

Geração

O programa irá gerar um número secreto aleatório dentro de um intervalo pré-definido (por exemplo, de 1 a 100).

Tentativa

Você deve digitar um número para tentar adivinhar qual é o número secreto.

Dicas

Após cada tentativa, você receberá uma dica para guiar seu próximo palpite:

  • Se o seu palpite for maior que o número secreto, você será avisado(a).
  • Se o seu palpite for menor que o número secreto, você também será avisado(a).

Contador

O número de tentativas que você fez será registrado para que você possa comparar seu desempenho.
